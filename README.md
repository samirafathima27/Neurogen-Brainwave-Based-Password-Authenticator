# Neurogen-Brainwave-Based-Password-Authentication System
NeuroGen is a machine learning-based authentication system that uses EEG (brainwave) signals as a biometric password. Unlike traditional passwords, this system leverages unique neural patterns to verify user identity.

In this project, I built an end-to-end pipeline for processing EEG signals, including data cleaning, noise reduction, feature extraction, normalization, and model training. The system was trained on 1000+ EEG records and improved authentication accuracy from 61% to 85% through iterative feature engineering and model tuning.

The key challenge was handling noisy and inconsistent EEG data. I addressed this by applying structured preprocessing techniques and optimizing feature selection to improve model reliability.

This project demonstrates the use of machine learning for real-world biometric authentication and highlights challenges in working with raw physiological data.
